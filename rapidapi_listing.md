Nombre: Spain Mobile Towers API
Descripción: Consulta antenas móviles en España por operador, tecnología, provincia, municipio o coordenadas. Datos públicos oficiales del geoportal MINETUR, actualizados diariamente.


Endpoints:
- GET /antenas → filtros: provincia, municipio, operador, tecnologia, page, limit
- GET /antenas/{id} → detalle
- GET /antenas/near?lat=&lon=&radio_m= → antenas cercanas
- GET /stats → resumen por tecnología y operador


Planes sugeridos:
- Free: 50 req/día
- Pro: 1000 req/mes — 9 €/mes
- Business: 10000 req/mes — 39 €/mes


Estrategia de lanzamiento:
- Promocionar en foros de desarrolladores, LinkedIn telecom, Product Hunt
- Ofrecer prueba gratuita a empresas para obtener testimonios y casos de uso
